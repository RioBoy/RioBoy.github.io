About

Perkenalkan, saya Januario Boy mahasiswa UNWIRA (Universitas Katholik Widya Mandira Kupang) jurusan Ilmu Komputer S1. Selain sebagai seorang mahasiswa saya juga berprofesi sebagai freelance, saya memiliki keahlian di bidang Web Development khusunya dalam Front-End. Dalam dua tahun terakhir saya memfokuskan diri untuk mengembangkan skill yang dimiliki, baik dalam soft skill seperti kemampuan berbicara dalam Bahasa inggris maupun dalam bidang hard skill yaitu pembuatan website.

Keahlian yang saya di kuasai diantaranya Web Design menggunakan tools seperti Adobe XD dan Figma, Front-End Development menggunakan HTML, CSS, JavaScript, Bootstrap dan Materiallize. Selain Front-End, saya juga menguasai beberapa Bahasa Pemrograman untuk menunjang bagian Back-End diantaranya PHP dengan Framework Codeigniter dan NodeJS dengan Framework ExpressJS. Semoga saya dapat membantu anda membuat website yang diinginkan, senang dapat membantu. Have a great day!
